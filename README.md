# 📖 About the Project

The **Product Wholesale System** is a Python-based inventory management application developed for **WeCare Skin Care Store** to simplify wholesale product management and inventory operations.

The application enables users to manage products, process wholesale sales, restock inventory, apply promotional offers, update stock levels automatically, and generate detailed invoices. Product information is stored in a text file (`products.txt`), ensuring inventory records remain accurate after every transaction.

The project demonstrates practical implementation of Python programming concepts including modular programming, file handling, dictionaries, lists, functions, exception handling, and input validation.

# ⚙️ Development Approach

The project follows a modular programming approach where each module has a dedicated responsibility.

### Main.py
Provides the menu-driven interface and serves as the application's entry point.

### Read.py
Reads product information from the inventory file and stores it for processing.

### Write.py
Writes updated inventory information back to the data file after every transaction.

### Operation.py
Contains the application's core functionality, including:

- Product Display
- Product Sales
- Buy 3 Get 1 Free Offer
- Inventory Restocking
- Stock Updates
- Invoice Generation
- Price Calculation

This modular architecture improves readability, maintainability, and scalability.
